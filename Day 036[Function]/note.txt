Slide 1: Title Slide
Title: Understanding JavaScript Functions

Subtitle: A Quick Guide of JS Function

Slide 2: What is a Function?
Definition: A function is a set of instructions that performs a task.
Purpose: Reuse code easily.

Slide 3: Creating a Function
Example:
function greet() {
  console.log('Hello, World!');
}
Parts: function keyword, name, and code block.

Slide 4: Using a Function
greet();
Action: Calls the function to run the code inside.

Slide 5: Function with Inputs

function add(a, b) {
  return a + b;
}

Inputs: a and b are parameters.

Slide 6: Arrow Function

const multiply = (a, b) => {a * b};
Arrow Function: Shorter way to write functions.

Slide 7: Conclusion
Recap: Functions help reuse code and make it easy to manage.
